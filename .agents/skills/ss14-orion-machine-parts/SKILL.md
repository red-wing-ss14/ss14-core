# Orion Machine Parts And Upgrades

Use this skill to route machine part quality/rating changes through the existing Orion construction and upgrade flow.

## Workflow

1. Open `references/machine-parts-map.md` before editing.
2. Inspect the existing component, construction graph step, upgrade event, and part exchanger path before adding behavior.
3. Check machine frames, construction graphs, lathe recipes, UI/feedback, and tests when upgrade behavior changes.
4. For TG parity tasks, inspect current Orion behavior first; compare with TG only after that.
